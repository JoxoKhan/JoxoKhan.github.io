Linguagens de marcação(HTML5, CSS3) e linguagem javascript. Conto ainda com uma revisão de conteúdo, corrigindo assim, alguns eventuais erros na escrita

Framework JQuery, facilidando e agilizando a criação de aplicações web usando pouco código.

A linguagem PHP é mundialmente conhecida e uma das mais utilizadas pela facilidade em aprendê-la, manuseá-la, além de ser compatível com quase todos os sistemas operacionais que existem – o que torna seu custo menor.

Angular é uma plataforma e framework para construção da interface de aplicações usando HTML, CSS e, principalmente, JavaScript, criada pelos desenvolvedores da Google. Ele possui alguns elementos básicos que tornam essa construção interessante.

Conto ainda com versionamento e controle de códigos através da ferramenta GIT e do sistema online GitHub, para melhor controle dos códigos dos projetos

Na maioria dos projetos, eu utilizo Bootstrap, um framework poderoso para estilizar páginas, e deixar tudo mais responsivo, adaptando-as assim para diversos dispositivos, sejam eles: desktops, mobiles ou tablets
